# House Of Lancers
> Premium Luxury Digital Growth & AI Automation Agency Website

Welcome to the official codebase for the **House Of Lancers** website. House Of Lancers is a luxury growth, marketing, and AI automation partner helping premium brands scale through digital transformation, creative design, and workflow automation.

---

## 🌟 Features of the Website

- **Premium Luxury Aesthetics**: Sleek dark/light theme options with automatic local storage saving and system preferences detection to prevent flash on load.
- **Dynamic Animations**: Implemented using GSAP (GreenSock Animation Platform) and ScrollTrigger for a smooth, high-end feel.
- **Interactive UI Components**:
  - **Custom Cursor & Noise Overlay** for a modern, immersive digital experience.
  - **Interactive Social Hub Switcher** showing reels, carousel, and paid ad metrics.
  - **Automated Operations Loop** mapping out lead routing stages interactively.
  - **Interactive ROI Calculator** allowing users to estimate monthly and yearly operational savings from automation.
- **10+ Curated Pages**: Services, SMM, Business Automation, Design, Video, Portfolio, About Us, Blog, Admin, and Contact.
- **Fully Responsive**: Crafted with mobile-first modern CSS Grid & Flexbox layouts.

## 🛠️ Getting Started & Running Locally

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 1. Install & Run Dev Server
This project uses `http-server` to run a local development environment. Simply execute:

```bash
npm run dev
```

This starts a server at:
* **[http://localhost:3000](http://localhost:3000)**

### 2. Static Method
Since this is a client-side static web application, you can also open the entry point [index.html](file:///Users/bhavinkuriya/Downloads/HOL%20Website/index.html) directly in any web browser.

## 🎨 Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES6)
- **Styling**: Modern Vanilla CSS (variables, grid, flexbox, custom scrollbars, backdrop filters)
- **Animation**: GSAP (TweenMax, TimelineMax) & ScrollTrigger CDN
- **Fonts**: Google Fonts (*Outfit* and *Plus Jakarta Sans*)
- **Icons/Graphics**: Custom inline SVGs & premium imagery assets
